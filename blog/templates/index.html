What is a client?

A client is an invidual computer that is accessing information and programs stored in a server.

What is a server?

A server stores and provides information to clients.

What is an HTTP request?

It is an information request message from a client to a server over HTTP. Some of the methods include GET, POST, PUT, DELETE.

What is an HTTP response?

An HTTP response is made from the server to client, where the server sends requested information back to the client.

what is Django and why do we use it?

Django is a web application framework, written in Python. It is used to create web apps.
